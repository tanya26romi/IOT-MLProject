# Project on Internet of Things
summer project
This project is based on IOT and ML using Raspberri pi. 

Hardware Used - 
- Raspberri Pi
- Power Bank
- Battery
- Xigbee Module
- Jump Wires
- Web Camera
- Motors and Gears

Software Used - 
- Operating System - Linux, Debian
- VNC Viewer
- Programming Language - Python
- Source Code Editor - Atom

Python Libraries - 
- OpenCV
- Numpy
- Xigbee module
